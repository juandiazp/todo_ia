# todo_ia
# 📝 To-Do Checklist App

## 📌 Descripción
Aplicación de gestión de tareas (To-Do Checklist) que permite a los usuarios crear, visualizar, completar y eliminar tareas de manera sencilla y eficiente.

El objetivo principal es ofrecer una solución rápida, intuitiva y escalable, siguiendo buenas prácticas de desarrollo y priorizando la experiencia de usuario.

---

## 🚀 Tecnologías utilizadas

- Flutter (Frontend)
- Firebase Firestore (Base de datos en tiempo real)

> Se eligió esta arquitectura para reducir tiempos de desarrollo sin sacrificar escalabilidad.

---

## ⚙️ Funcionalidades

- ✅ Crear nuevas tareas
- 📋 Listar tareas en tiempo real
- ✔️ Marcar tareas como completadas
- ❌ Eliminar tareas
- 🔄 Persistencia de datos en la nube

---

## 🧱 Modelo de datos

```json
Task {
  id: string,
  title: string,
  completed: boolean,
  createdAt: timestamp,
  dueDate: timestamp (optional)
}
```
